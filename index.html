What Is This Program Used For?

This program was developed to securely store private documentation in two separate locations:

  • WebDAV-based cloud storage
  • Locally on the user’s computer

To use this program, you must have access to a WebDAV cloud storage, which can be either free or paid.

Connecting to Yandex.Disk via WebDAV

  1. Register an account on yandex.ru.
  2. Create an application password for WebDAV access: Yandex App Passwords.
  3. Launch the Atlaskey program.
  4. On first launch, create a master key for the secure database. This key will serve as the password for your encrypted ZIP archives.
    Stored at: documents\atlaskey\config.zip
  5. Go to the Tools tab and install required components by following on-screen instructions.
  6. Go to the WebDAV tab and enter your connection details:
    • WebDAV URL: https://webdav.yandex.ru
    • User: your Yandex username (part of email before @yandex.ru)
    • Password: WebDAV application password
  7. Click Check Connection. If successful, the button will turn green.
  8. Upon success, a folder named cryptdav is created in your WebDAV cloud for encrypted documents.
  9. Use buttons:
    Mount WebDAV – mount the storage
    Unmount WebDAV – unmount the storage
    • Windows: appears as drive (e.g., X:), auto-assigns another letter if taken
    • Linux/macOS: mounted to documents\atlaskey\cryptdav

Managing Document Backups

To manage backups of your documents in the WebDAV cloud, go to the Documents tab.
Here you can perform the following five operations:

  1. Create backups in ZIP format (protected with master key) at documents\atlaskey\backup
  2. Restore files from a backup to WebDAV cloud
  3. Delete files/folders from WebDAV cloud and backups
  4. Rename files
  5. Create new folders/documents (built-in editor included)

Data Recovery Scenarios

Scenario 1: WebDAV access lost

  1. Register a new account and create a new WebDAV app password.
  2. Update User/Password in Atlaskey WebDAV tab.
  3. Connect to new cloud; folder cryptdav will be empty.
  4. Go to the Documents tab and select Recovery.
  5. Data restored from backup to new WebDAV cloud.

Scenario 2: Lost PC with backups

Note: Keep documents\atlaskey\config.zip separately (USB, phone, another PC, cloud).

  1. Take new computer.
  2. Download Atlaskey from this site.
  3. Launch program; documents\atlaskey\ is created.
  4. If config.zip saved, place it in this folder and restart app; otherwise create new master key.
  5. Connect to existing WebDAV cloud.
  6. Go to the Documents tab and create new backup from cloud to local PC.
  7. All data restored and available locally.
